# ETEC_FINAL

## 📌 Project Overview

ETEC_FINAL is a PHP-based project that incorporates **Swiper.js** and **Bootstrap** to create a horizontally scrollable card layout. This project is designed to display multiple items in a single row, allowing users to scroll through the content interactively.

---

## 🏗 Project Structure

```
ETEC_FINAL/
│── asset/                 # Static assets (images, icons, etc.)
│── node_modules/          # Installed npm packages
│── public/                # Publicly accessible files
│── routers/               # Routing logic (if applicable)
│── src/                   # Source code
│   ├── controllers/       # Handles backend logic
│   │   └── main.js        # Main JavaScript file for Swiper.js
│   ├── models/            # Handles data models (if required)
│   ├── style/             # CSS styles
│   │   └── style.css      # Main stylesheet
│   ├── views/             # Frontend views
│   │   └── index.php      # Main view file with Swiper.js implementation
│── .gitignore             # Git ignore file
```

## 🚀 Technologies Used

- **PHP** - Backend scripting
- **Swiper.js** - Smooth scrolling and navigation
- **Bootstrap** - UI components and responsive grid system

## 📜 How to Run the Project

### **1️⃣ Install Dependencies**

```sh
npm install
```

### **2️⃣ Start Local PHP Server**

```sh
php -S localhost:8000 -t public
```

### **3️⃣ Open in Browser**

Go to: [http://localhost:8000/src/views/index.php](http://localhost:8000/src/views/index.php)
